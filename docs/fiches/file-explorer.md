Fiche documentaire pour l Explorateur de fichiers Windows.

Documentation sheet for Windows File Explorer.

## Francais

- ID: `file-explorer-fr`
- Titre: Explorateur de fichiers - Essentiels
- Applications: `explorer.exe`

### Navigation

| ID | Action | Touches | Description | Niveau |
| --- | --- | --- | --- | --- |
| `navigation-ouvrir-l-explorateur` | Ouvrir l explorateur | `Win` + `E` | Ouvre une fenetre de fichiers. | Essentiel |
| `navigation-nouvelle-fenetre` | Nouvelle fenetre | `Ctrl` + `N` | Ouvre une nouvelle fenetre. | Courant |
| `navigation-nouvel-onglet` | Nouvel onglet | `Ctrl` + `T` | Ouvre un onglet si disponible. | Courant |
| `navigation-fermer-onglet` | Fermer onglet | `Ctrl` + `W` | Ferme l onglet ou la fenetre. | Courant |
| `navigation-dossier-parent` | Dossier parent | `Alt` + `Up` | Remonte d un dossier. | Courant |
| `navigation-retour` | Retour | `Alt` + `Left` | Revient a l emplacement precedent. | Courant |
| `navigation-avancer` | Avancer | `Alt` + `Right` | Passe a l emplacement suivant. | Courant |
| `navigation-barre-d-adresse` | Barre d adresse | `Ctrl` + `L` | Active la barre d adresse. | Courant |
| `navigation-actualiser` | Actualiser | `F5` | Recharge le dossier. | Courant |

## English

- ID: `file-explorer-en`

### Navigation

| ID | Action | Touches | Description | Niveau |
| --- | --- | --- | --- | --- |
| `navigation-ouvrir-l-explorateur` | Open File Explorer | `Win` + `E` | Open a file window. | Essential |
| `navigation-nouvelle-fenetre` | New window | `Ctrl` + `N` | Open a new window. | Common |
| `navigation-nouvel-onglet` | New tab | `Ctrl` + `T` | Open a tab when supported. | Common |
