# Navigateurs / Browsers

Fiche documentaire unique pour Edge, Chrome, Firefox et Brave.

Single documentation sheet for Edge, Chrome, Firefox, and Brave.

## Sources

- https://support.microsoft.com/en-au/microsoft-edge/keyboard-shortcuts-in-microsoft-edge-50d3edab-30d9-c7e4-21ce-37fe2713cfad
