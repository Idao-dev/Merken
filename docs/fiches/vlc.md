# VLC

Fiche documentaire pour les raccourcis VLC utiles sur Windows.

Documentation sheet for useful VLC shortcuts on Windows.

## Sources

- https://thresh.videolan.me/vlc-user/userguide/hotkeys.html

## Francais

- ID: `vlc-fr`
- Titre: VLC - Essentiels
- Applications: `vlc.exe`

### Lecture

| ID | Action | Touches | Description | Niveau |
| --- | --- | --- | --- | --- |
| `lecture-lecture-pause` | Lecture pause | `Space` | Lance ou met en pause. | Essentiel |
| `lecture-stop` | Stop | `S` | Arrete la lecture. | Courant |
| `lecture-suivant` | Suivant | `N` | Passe au media suivant. | Courant |
| `lecture-precedent` | Precedent | `P` | Revient au media precedent. | Courant |
| `lecture-plein-ecran` | Plein ecran | `F` | Bascule le plein ecran. | Essentiel |
| `lecture-muet` | Muet | `M` | Coupe ou retablit le son. | Essentiel |
| `lecture-temps` | Temps | `T` | Affiche le temps. | Avance |

| `navigation-image-suivante` | Image suivante | `E` | Avance image par image. | Expert |
