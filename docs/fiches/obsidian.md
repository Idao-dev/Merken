## Francais

- ID: `obsidian-fr`
- Titre: Obsidian - Essentiels
- Applications: `obsidian.exe`

### Navigation

| ID | Action | Touches | Description | Niveau |
| --- | --- | --- | --- | --- |
| `navigation-palette-commandes` | Palette commandes | `Ctrl` + `P` | Ouvre la palette. | Essentiel |
| `navigation-ouverture-rapide` | Ouverture rapide | `Ctrl` + `O` | Recherche une note. | Essentiel |
| `navigation-nouvelle-note` | Nouvelle note | `Ctrl` + `N` | Cree une note. | Essentiel |
| `navigation-fermer-onglet` | Fermer onglet | `Ctrl` + `W` | Ferme l onglet. | Courant |
| `navigation-onglet-suivant` | Onglet suivant | `Ctrl` + `Tab` | Passe a l onglet suivant. | Courant |
| `navigation-onglet-precedent` | Onglet precedent | `Ctrl` + `Shift` + `Tab` | Revient a l onglet precedent. | Courant |
| `navigation-parametres` | Parametres | `Ctrl` + `,` | Ouvre les parametres. | Avance |
