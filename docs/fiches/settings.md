# Parametres Windows / Windows Settings

Fiche documentaire pour l application Parametres Windows.

Documentation sheet for Windows Settings.

## Sources

- https://support.microsoft.com/en-us/windows/keyboard-shortcuts-in-apps-139014e7-177b-d1f3-eb2e-7298b2599a34

## Francais

- ID: `settings-fr`
- Titre: Parametres Windows - Essentiels
- Applications: `systemsettings.exe`

### Navigation

| ID | Action | Touches | Description | Niveau |
| --- | --- | --- | --- | --- |
| `navigation-ouvrir-les-parametres` | Ouvrir les parametres | `Win` + `I` | Ouvre Parametres. | Essentiel |
| `navigation-retour` | Retour | `Backspace` | Revient a la page precedente. | Courant |
| `navigation-element-suivant` | Element suivant | `Tab` | Passe au controle suivant. | Courant |
| `navigation-element-precedent` | Element precedent | `Shift` + `Tab` | Revient au controle precedent. | Courant |
| `navigation-valider` | Valider | `Enter` | Active le controle selectionne. | Courant |
| `navigation-basculer-option` | Basculer option | `Space` | Change une case ou un bouton. | Courant |
