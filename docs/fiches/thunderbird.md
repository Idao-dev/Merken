# Thunderbird

Fiche documentaire pour les raccourcis Thunderbird utiles sur Windows.

Documentation sheet for useful Thunderbird shortcuts on Windows.

## Sources

- https://support.mozilla.org/bm/kb/keyboard-shortcuts-thunderbird

## Francais

- ID: `thunderbird-fr`

- Applications: `thunderbird.exe`

| `messages-repondre-a-tous` | Repondre a tous | `Ctrl` + `Shift` + `R` | Repond a tous. | Courant |
| `messages-transferer` | Transferer | `Ctrl` + `L` | Transfere le message. | Courant |
| `messages-archiver` | Archiver | `A` | Archive le message. | Courant |
| `messages-supprimer` | Supprimer | `Delete` | Supprime le message. | Courant |
| `messages-lu-non-lu` | Lu non lu | `M` | Change l etat lu. | Courant |
| `messages-indesirable` | Indesirable | `J` | Change le statut indesirable. | Avance |
